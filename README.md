## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/DevTruce/natours.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Dev environment
   ```sh
   npm run start
   ```
4. Build package
   ```sh
   npm run build:css
   ```
